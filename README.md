# 📊 Análise Exploratória de Dados – COVID-19

## Objetivo
Analisar dados da COVID-19 para identificar padrões de evolução da pandemia, comparar indicadores entre países e gerar visualizações que auxiliem na interpretação dos dados.

## 📁 Dataset
Fonte: Kaggle – COVID-19 Clean Complete Dataset

## 🛠️ Tecnologias utilizadas:
  - Python
  - Pandas
  - Matplotlib
  - Google Colab

## 🔍 Perguntas respondidas
  - Como evoluíram os casos globais ao longo do tempo?
  - Quais países apresentaram maior número de casos confirmados?
  - Quais países apresentaram maior número de mortes?
  - Qual a relação entre casos confirmados, mortes e recuperações?
  - Como foi a evolução dos casos no Brasil?

## 📈 Principais insights
  - Estados Unidos e Brasil lideram o ranking de casos e mortes no período analisado.
  - Observa-se crescimento contínuo dos casos globais e no Brasil ao longo do tempo.
  - Existe forte correlação entre casos confirmados e número de óbitos.
