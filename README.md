# 📊 Análise Exploratória de Dados – COVID-19

## 📌 Sobre o projeto
  Este projeto foi desenvolvido com o objetivo de aplicar técnicas de Análise Exploratória de Dados (EDA) em um conjunto de dados reais sobre a pandemia de COVID-19.
  A análise busca identificar tendências de evolução da doença, comparar indicadores entre países e gerar insights por meio de visualizações construídas em Python.


## 🎯 Objetivos
  - Analisar a evolução dos casos de COVID-19 ao longo do tempo;
  - Comparar indicadores entre diferentes países;
  - Avaliar a relação entre casos confirmados, recuperações e óbitos;
  - Explorar o comportamento da pandemia no Brasil;
  - Aplicar técnicas de limpeza, transformação e visualização de dados.


## 📁 Dataset
  Fonte: Kaggle – COVID-19 Clean Complete Dataset

  O conjunto de dados contém informações sobre:
  
  - Casos confirmados;
  - Óbitos;
  - Recuperações;
  - Datas de registro;
  - Países e regiões afetadas.


## 🛠️ Tecnologias utilizadas
  - Python
  - Pandas
  - Matplotlib

  
## 💡 Competências demonstradas
  - Limpeza e tratamento de dados
  - Manipulação de dados com Pandas
  - Análise Exploratória de Dados (EDA)
  - Construção de visualizações
  - Interpretação de indicadores
  - Geração de insights
  - Storytelling com dados


## 🔍 Perguntas respondidas
  - Como evoluíram os casos globais ao longo do tempo?
    Análise da evolução temporal dos casos confirmados em escala mundial.
    
  - Quais países apresentaram maior número de casos confirmados?
    Identificação dos países mais impactados pela pandemia no período analisado.
    
  - Quais países apresentaram maior número de mortes?
    Comparação entre países considerando o número total de óbitos registrados.
    
  - Qual a relação entre casos confirmados, mortes e recuperações?
    Avaliação dos principais indicadores da pandemia e suas correlações.
    
  - Como foi a evolução dos casos no Brasil?
    Análise específica do comportamento da pandemia em território nacional.


## 📈 Principais insights
  - Os dados mostram concentração significativa dos casos em poucos países durante o período analisado.
  - Estados Unidos e Brasil figuram entre os países com maior volume de casos confirmados e óbitos.
  - O crescimento acumulado apresentou comportamento acelerado ao longo do período observado.
  - Foi identificada forte associação entre o número de casos confirmados e o volume de óbitos registrados.
  - A evolução dos indicadores brasileiros acompanhou a tendência global de crescimento observada no dataset.
